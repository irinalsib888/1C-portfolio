# Задание «Общие положения» (автоматизация торговой организации)

## Источник
Курс по 1С:Предприятие 8.2 / 8.3.  

## Суть задания
Необходимо автоматизировать торговую организацию:
- Кадровый учёт (сотрудники, квалификация, дети, кадровые приказы).
- Закупка и продажа товаров, услуги.
- Учёт поставщиков, покупателей, договоров.
- Взаиморасчёты (приходные/расходные накладные, оплата).
- Валютный учёт (опционально, подключается).
- Отчёт: продажи в разрезе сотрудников за текущий месяц.

## Что уже сделано
- Создана информационная база.
- Созданы справочники «Сотрудники», «Контрагенты» и др, документы, регистры сведений, отчет.

## Что планируется
- Реализация всех объектов по ТЗ.
- Формирование печатной формы для расходной накладной.
- Разработка отчёта по продажам.

## Как проверять
- Открыть конфигурацию в режиме 1С:Предприятие.
- Посмотреть созданные справочники, документы, отчёты.

# Работы по книге Радченко – Хрусталева «1С:Предприятие 8.3. Практическое пособие разработчика»

## О книге
- **Авторы:** М.Г. Радченко, Е.Ю. Хрусталева  
- **Издание:** 3-е  
- **Содержание:** примеры и типовые приёмы разработки на платформе 1С:Предприятие 8.3.

## Что здесь представлено
В этой папке собраны конфигурации и файлы, созданные мной в процессе **пошагового выполнения заданий** из книги. Задания выполнялись в учебных целях для освоения ключевых механизмов платформы.

### Охваченные темы (по занятиям)
- **Занятие 3.** Справочники  
- **Занятие 4.** Документы  
- **Занятие 6.** Регистры накопления  
- **Занятие 7.** Простой отчёт  
- **Занятие 9.** Периодические регистры сведений  
- **Занятие 11.** Проведение документа по нескольким регистрам  
- **Занятие 12.** Оборотные регистры накопления  
- **Занятие 13.** Отчёты (в т.ч. СКД)  
- **Занятие 14.** Оптимизация проведения документа  
- … и другие (по мере выполнения)

### Формат выполнения
Каждое задание выполнялось по инструкциям пособия, что позволило:
- закрепить теорию на практике;
- понять структуру типовых конфигураций;
- освоить работу с регистрами, отчётами, документами и другими объектами.

## Как это связано с моим обучением
Книга является одним из основных источников при изучении платформы. Выполнение всех занятий дало мне системное понимание разработки в 1С:Предприятие 8.

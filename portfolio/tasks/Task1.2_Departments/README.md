# Задача 1.2 — Подразделения и подчинение сотрудников

## Условие
1. Создать справочник «Подразделения» с реквизитом «Телефон».
2. В справочник «Сотрудники» добавить реквизит «Подразделение».
3. Реализовать автозаполнение телефона подразделения в форме элемента сотрудника при выборе подразделения.
4. Вывести колонку «Подразделение» в форме списка сотрудников.

## Решение

### Созданные и доработанные объекты
- Справочник `гр_Подразделения` с реквизитом `гр_Телефон` (тип «Строка», длина 17) задана маска.
- Доработан справочник `гр_Сотрудники` 

### Реализованный функционал
- **Форма элемента сотрудника:** при выборе подразделения автоматически подставляется его телефон в соответствующее поле (если телефон заполнен в карточке подразделения). Реализовано в обработчике события `ПриИзменении` реквизита `Владелец` (см. скриншот `task1.2_code.png`).
- **Форма списка сотрудников:** добавлена колонка «Подразделение» для наглядности.
